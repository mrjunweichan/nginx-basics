  
  

## Web Server
  
**Configure Web Server with /etc/nginx/conf.d/web.conf**  
```bash
cat <<EOF >>/etc/nginx/conf.d/web.conf
server {
    
    listen       9001;
    index  index.html;
    
    location / {
        root   /opt/services/App1;
    }
}

server {
    
    listen       9002;
    index  index.html;
    
    location / {
        root   /opt/services/App2;
    }
}

server {
    
    listen       9003;
    index  index.html;
    
    location / {
        root   /opt/services/App3;
    }
}
EOF
```
  
**Reload NGINX**  
```sh
nginx -t
nginx -s reload
```
