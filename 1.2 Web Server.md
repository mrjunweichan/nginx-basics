  
  

## Web Server
  
**Configure Web Server**  
```nginx
sudo vim /etc/nginx/conf.d/web.conf
```
  
  
**/etc/nginx/conf.d/web.conf**  
```nginx
server {
    
    listen       9001;
    index  index.html;
    
    location / {
        root   /opt/services/App1;
    }
}

server {
    
    listen       9002;
    index  index.html;
    
    location / {
        root   /opt/services/App2;
    }
}

server {
    
    listen       9003;
    index  index.html;
    
    location / {
        root   /opt/services/App3;
    }
}
```
  
**Reload NGINX**  
```sh
nginx -t
nginx -s reload
```
